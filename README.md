## 2 你认为脚手架除了为我们创建项目结构，还有什么更深的意义
  脚手架减少了不必要的重复工作，提高了生产力，同时也减少了人为操作带来的失误，通过约定规范了开发人员的主观行为，屏蔽个体的差异性，
  有利于团队的合作。
